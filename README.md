## 🛠️ How It Works

This catalog is powered by a proprietary **Automation Sentinel** (GitHub Actions + Python) that performs the following sequence every 24 hours:

1.  **Scanning:** Queries the GitHub API for all owned, forked, starred, and contributed repositories.
2.  **Categorization:** Applies a hierarchical logic (Owned > Forked > Starred > Contributed) to ensure every entry is precisely filed.
3.  **Synthesis:** Updates the README and dynamic badges, then commits the changes back to the core.

### 🔌 Tech Stack
- **Engine:** Python 3.10
- **Orchestrator:** GitHub Actions
- **Data Source:** GitHub REST API v3
- **Styling:** Dynamic Markdown & Shields.io

## 🚀 Want Your Own?

The **Living Repo Catalog** is designed to be extensible. To deploy your own instance:

1.  **Fork** this repository.
2.  **Update** `config.json` with your GitHub username.
3.  **Ensure** your workflow permissions allow writing back to the repository.
